# go-trainee-assignment
## Протокол

Описан в файле [schema.yaml](schema.yaml) и [schema.json](schema.json).

## Как запустить (с докером)

создать .env из .env.example
```bash
cp .env.example .env
```

```bash
docker-compose up --build
```

## Остановить приложение

```bash
docker-compose-down
```
